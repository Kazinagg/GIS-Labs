# Лабораторная работа №5: Локализация приложения

## Тема
Интернационализация и локализация интерфейса пользователя в среде RAD Studio.

## Цель работы
Реализовать поддержку руского языков в приложении.

## Инструменты
* **Среда разработки:** RAD Studio 12/13 (Delphi / FireMonkey).

## Описание процесса
В ходе выполнения работы проект был подготовлен к локализации. Процесс включал:
Перевод названий кнопок, меток (Labels), заголовков форм и системных сообщений.
